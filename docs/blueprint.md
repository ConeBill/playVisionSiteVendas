# **App Name**: Blush & Ink

## Core Features:

- BFF Integration Engine: A robust Backend-for-Frontend using adapters to normalize data from the external Bearer Token API into a unified MongoDB domain model.
- AI Desk Stylist Tool: An AI-powered tool that analyzes a photo or text prompt of a user's workspace to suggest perfectly matching stationery products from the catalog.
- Abstracted Domain Components: UI components built against local domain interfaces, ensuring zero disruption if the external commerce API source is swapped.
- Mercado Pago Pro Checkout: Full payment lifecycle management supporting PIX, credit cards, and automated order status synchronization.
- Predictive SEO Engine: Next.js optimized metadata using React Helmet to generate high-performance OG tags and structured data for each SKU.
- Resilient Cart Cache: State-driven cart management using Zustand with persistent hydration and offline support via PWA service workers.
- Customer Insights Hub: A centralized dashboard for order history, favorited stationery items, and secure JWT-based profile management.

## Style Guidelines:

- Primary Color: Deep Rose (#C74B82), providing strong contrast and sophisticated flair against a light scheme.
- Background Color: Pearl Pink (#FCF6F9), a soft, desaturated variation of the primary hue for a high-end stationery feel.
- Accent Color: Lavender Dusk (#A54BC7), an analogous hue that provides visual distinction for call-to-action elements.
- Headline Font: 'Belleza', a humanist sans-serif with art-focused personality. Body Font: 'Alegreya', an elegant serif perfect for the literary nature of stationery content.
- Thin-stroke, minimalist vector icons with rounded terminals to mirror the 'clean' Shopify aesthetic.
- Airy, 'Bento-style' grid with significant white space, subtle 16px border-radii, and a focus on high-fidelity product imagery.
- Micro-interactions using Framer Motion, specifically soft 'spring' transitions on hover and fade-in loading states for search results.
